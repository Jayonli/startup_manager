# 自启动管理器

一个简单的图形界面工具，用于管理Windows系统中的自启动软件。用户可以通过拖拽或选择文件的方式，将需要自启动的软件添加到系统启动项中。

## 功能特点
- **图形化界面**：简单易用的界面，支持拖拽和选择文件。
- **自启动管理**：将软件添加到Windows的“启动”文件夹，实现开机自启动。
- **快捷方式创建**：自动为选中的软件创建快捷方式，无需手动操作。

## 使用方法

### 1. 安装依赖
确保你已安装Python环境（推荐Python 3.8及以上版本）。运行以下命令安装所需的依赖库：
```bash
pip install pywin32
```

### 2. 运行程序
将脚本保存为startup_manager.py，然后通过以下方式运行程序：
```
pythonw startup_manager.py
```
- 生成可执行文件：
使用PyInstaller将脚本打包为可执行文件：
```
pip install pyinstaller
pyinstaller --onefile --windowed startup_manager.py
```
打包完成后，运行dist/startup_manager.exe即可。

### 3. 添加自启动软件
打开程序后，点击“选择文件”按钮，选择需要自启动的软件（.exe文件）。
也可以将软件快捷方式拖拽到程序窗口中（需要额外支持拖拽功能）。
点击“保存到自启动”按钮，将选中的软件添加到Windows的自启动项中。

### 4. 删除自启动软件
在程序的列表框中选中需要删除的软件路径。
点击“删除选中”按钮，从列表中移除该路径。

### 5.注意事项
权限问题：某些操作（如写入自启动文件夹）可能需要管理员权限。
路径问题：确保添加的文件路径正确，且文件存在。
安全性：在添加自启动软件时，请确保其来源可靠，避免运行恶意软件。

扩展功能
如果需要支持拖拽功能，可以安装tkinterdnd2库：
```
pip install tkinterdnd2
```
